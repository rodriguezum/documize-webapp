Documize Secure Spaces

Document management tools are great for sharing information, letting multiple individual and teams edit and collaborate, but when it comes to security most tools are either incredible complex to manage or have too little security.
For most uses we end up having:

  • Internal documents for finance, marketing, operations
  • Internal Wikis with procedures for development, customer service
  • External Wikis for the clients or prospects
  • Individual documents with proposals that go out to clients

We believe that all of these different uses cases and tools are best served under one tool, for this we have created Documize, realizing we can integrate all these tools under heavy but flexible security areas where teams can work individually or together, for that we have build

Spaces

Spaces are areas that allow for documents to be completely separate documents from any other areas, these spaces are usually for entire departments or whole companies. Finance, Operations, Marketing departments can all have secure spaces where their individual files are kept within the team, if we wish to further separate, protect and organize files within spaces we use Categories

Categories

Categories allow for multiple secure areas within a space and allow for multiple teams to work together and dynamically removing and adding them when you need to.
Users and teams are assigned per space and per category so we can manage a document creation process as simple as you add a trello card on your project management list, but securely manage who gets to see that card and what gets done there.

What is the main difference between categories and spaces?
One document can live in multiple categories but only belong only to one space, this allows for documents to be added to several categories such as:

  • Backlog
  • Todo
  • Sprint X
  • In Progress
  • Approval/QA
  • Completed
  • ...

And at the same time for instance belong to other categories such as:

  • Marketing
  • Design
  • Legal
  • Finance
  • Copywriting
  • ...
